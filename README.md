# n8n-Moderate-Facebook-group-posts-with-OpenAI-Slack-and-Airtable
Moderate Facebook Groups automatically with this **n8n workflow automation**. It analyzes new posts using AI, detects spam and policy violations, logs incidents in Airtable, alerts moderators in Slack, and automatically hides high-severity content—making community management faster, safer, and more consistent with **n8n**.
